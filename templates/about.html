AreumSec — Real-Time Threat Intelligence & Analysis

AreumSec is a real-time web-based cybersecurity threat analysis application built with Python Flask and Bootstrap. Designed for security analysts and VAPT professionals, it offers a clean, intelligence-agency-inspired interface for rapid file/file hash, URL and IP investigations.

Developer

Pranay Wajjala

Features

  • Real-Time Analysis
    URLs & Domains, IPv4 / IPv6 addresses, File Hash (SHA256), EML / MSG / TXT files
  • VirusTotal + Google SafeBrowsing Integration
    API keys are securely managed in a separate config file.
  • DNSBL Check & Logging
    A separate list of DNS Blacklist.
  • Offline First
    Lightweight and self-hosted — ideal for closed environments or air-gapped labs.

Tech Stack

  • Python 3
  • Flask (Backend)
  • Bootstrap 5 (Frontend)
  • SQLite (Separate DB for DNSBL results)
  • VirusTotal & Google SafeBrowsing APIs
